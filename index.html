prazer, eu sou o Ângelo

🌟 Meu nome é Ângelo, eu sei criar sites e sou autista.

"Meu jeito autista de pensar traz detalhes únicos para meus códigos — se algo parecer diferente, é por design 😊"

Tenho paixão por transformar ideias em páginas funcionais e bonitas. Uso HTML, CSS e JavaScript para criar projetos que comunicam e encantam. Acredito que a web deve ser um lugar acessível para todos, e cada linha de código é uma oportunidade de melhorar.

💬 Quando fizer algo errado, me falem. Estou sempre aberto a feedbacks e adoro aprender com quem me acompanha.
